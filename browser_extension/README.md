# 浏览器插件

## 安装方法

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `browser_extension` 文件夹

## 使用方法

### 1. 启动服务器

```bash
cd server
python server.py
```

### 2. 配置插件

1. 打开插件弹窗
2. 输入服务器地址（格式：`ws://IP:PORT`）
3. 点击「开始录制」

### 3. 播放视频

在B站、YouTube或其他网站播放视频，音频会被实时转写。

## 服务器地址格式

| 类型 | 地址示例 |
|------|--------|
| 本地 | `ws://localhost:8000` |
| 局域网 | `ws://你的IP:8000` |
| 公网 | `ws://服务器IP:8000` |

## 权限说明

插件需要以下权限：
- `tabCapture` - 捕获标签页音频
- `activeTab` - 访问当前标签页

## 注意事项

1. 部分视频可能有DRM保护，无法捕获
2. 需要保持标签页处于激活状态
3. 服务器需要GPU支持
